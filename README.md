# SolidWelcomeLogin

To run locally:
1. Clone the Repository.
2. Install the node dependencies using:
                ```npm ci```
4. Generate the webpack bundle using: 
            ```npm run build```
6. Start the server using: 
            ```npm start```

App is hosted live on [github](https://sindhu-vasireddy.github.io/SolidLoginGreeter/)
